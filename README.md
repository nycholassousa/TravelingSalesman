# Traveling Salesman

## Atividade

Implementar solução PD para o Problema do Caixeiro viajante

## Compilando

Supondo que você esteja no mesmo diretório, simplesmente invoque make

```sh
$ make
```

## Executando o Código

```sh
$ ./salesman SAMPLE_PATH
```

Exemplo: 

```sh
$ ./salesman samples/pvc04.txt
```
